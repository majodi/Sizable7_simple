# Sizable7 simple
Simple Sizable7 example for just one module

Sizable7 is a large (185mm x 135mm / 7.3" x 5.3") addressable RGB 7-segment display. For more info see [sizable7.nickstick.nl](https://sizable7.nickstick.nl/).
